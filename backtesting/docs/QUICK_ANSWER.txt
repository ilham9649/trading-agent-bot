Q: When I run:
   python run_backtest.py --symbol AAPL --start 2024-07-01 --end 2024-10-01
   
   Does the bot trigger once or several times?

A: The bot triggers SEVERAL TIMES - specifically 64 TIMES!
   (Once for each trading day in the period)

================================================================================
                         WHAT HAPPENS
================================================================================

July 1 - October 1, 2024 = 92 calendar days
                          = 64 TRADING days (excluding weekends/holidays)

For each of the 64 trading days:
  
  Day 1 (2024-07-01):
    ├─ Fetch price: $210.62
    ├─ Run TradingAgent analysis (2-3 minutes)
    │  ├─ Get news from 2024-07-01
    │  ├─ Get sentiment from 2024-07-01
    │  ├─ Calculate indicators as of 2024-07-01
    │  └─ Make decision: BUY/SELL/HOLD
    └─ Execute trade if conditions met
  
  Day 2 (2024-07-02):
    ├─ Fetch price: $212.34
    ├─ Run TradingAgent analysis (2-3 minutes)
    └─ Make decision and execute
  
  ... continues for ALL 64 days ...
  
  Day 64 (2024-09-30):
    ├─ Fetch price: $233.00
    ├─ Run TradingAgent analysis (2-3 minutes)
    └─ Make decision and execute
